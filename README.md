# backend

## Initialize
run:
- `python -m venv venv`
- `source venv/bin/activate`
- `pip install -r requirements.txt`

## Start server
- `./runner.sh`

Server should be available at http://localhost:5000

### windows
run:
- `python -m venv venv`
- `venv\Scripts\activate`
-`pip install -r requirements.txt`
### start server
- python -m app.main
